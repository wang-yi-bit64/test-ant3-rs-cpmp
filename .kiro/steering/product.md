# Product

A React-based date picker component library featuring multi-date selection capabilities. The project provides custom date picker components built on top of Ant Design v3, with support for non-consecutive date selection, keyboard navigation, and configurable selection limits.

## Key Features

- Multi-date selection with visual feedback
- Keyboard navigation support (arrow keys, Ctrl+arrows for month navigation)
- Maximum selection count limits
- i18n support (zh-CN, en-US)
- Compatible with Ant Design v3 styling
